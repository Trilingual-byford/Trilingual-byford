６月６日
```
せっかく日曜日なのでカラオケに行くと決めたんけど、午後買い物終わってからBANBANのカラオケ店に行くと「申し訳ない、今満室になっています、これから一時間後部屋が空きかもしれません」と言われた。
まあ、いいか、先買った野菜とか持ってカラオケ室に入ってちょっと恥ずかしいと思うので直ちに帰ると決めました。
歩行で住むところに到着。時間的余裕なのでご飯を作ります。さしぶりに火を使って料理を作るのでまずは簡単な、昔からよく作ったトマトと卵の炒めを作ってみます。
日本の中華料理屋で食べたマトと卵の炒めはネギが使わなかった、私の経験を元にすると葱はマトと卵の炒めの決め手を言っても過言ではない。今回は葱を最後に入れてそうめんと混ぜって食べました。
この程度の努力でこんな美味しさの料理を作って満足した。

6月13日
```
土曜日の夜、昼寝をしていた時、急にお母さんからビデオ通話の請求が来ました、何が発生したかと思いながら携帯のオーケーボタンを押してビデオ通話に出ました。
向こうはお母さんとお姉さんの顔が映っています、みんな心配な顔をしていた。お母さんは「祖母の健康状況はかなり厳しい、今日は最後のチャンスかもしれません、祖母に何か言いたいことが有れば、
今のうちに」と言った。
恥ずかしいことかもしれませんが、祖母の病弱な顔が携帯に映されていた時、僕の頭が真っ白で何を言うが全然わからなかった。人って死んでいる人に対して何を言っていいですか？いってらっしゃいとか言ったら
不孝と思われますか、その場で悲しい気持ちとかありませんでした。
晩ご飯はラメンーにすると決めたので、駅付近のラメンー屋に行く途中、田舎で祖母が植えたメロンを食べたあの夏を思い出した、祖母はとても誠実で勤勉な人です。毎回田舎に帰ったこと、祖母と祖父が植えた
新鮮な果物と野菜を楽しむことができます、歳をとったとしても休まない、美味しい果物と野菜を植えて鴨とか鶏とかを飼育する。この勤勉さは誰にも敵わない
祖母は苦痛なくあの世界に行くように。

6ヶ月20日
```
昨日一日雨降りました、今日は晴れ晴れなので水泳に行きます。初めて日本で水泳なので水着も眼鏡も帽子も持っていないので先に付近のモールで買いに行くと決めましたが、
LaLaPortというモールの一階と二階の全ての店をチェックしたけど、どこにも水着を販売しませんでした。今日水泳多分できないと思いますけど、一応ご飯を食べて体育Centerにでまた様子を見た方がいいと決めたんだ。
体育せんーたに到着、受け付けところの付近の自動販売機で水泳用の眼鏡を発見！購入して受け付けのところで今日3時からのプール入場券を取りに行った。
１０分ぐらいに列に並んでたらやっと入場が始まった。入場したらまずロッカールームに入る、男専用のロッカールームだけと、なんとお父さんが連れた幼い女の子も入っている。
ロッカの使い方が知らないので勝手に50円玉を入れて鍵を取ってロッカをかけて開けったらロッカ一回分の使い機会がなくした、もう一枚入れないとロッカをロックできない。
プール入場後、最初プールに入った時、プール周り見ているスタフーさんに水泳帽子を被っていない状態でプールに入っちゃぁだめと言われました。スイムシャップが持ってないと言うたら
スタフーさんは「じゃぁ、プールに入っちゃだめ」って返事した、その時、帰るしかないと思っていた、帰る前にまず一旦退場して外の自動販売機をチェックしてそこでスイムキャップを購入できたら
すぐ帰るではなく引き続きここで遊べるかもしれない。そして外の自動販売機の売り物を確認したら600一枚のスイムキャップが販売されていたのでそれを買ってもう一度プールに入場しました
プールにいるなのスタッフさんが意地悪いね、そっとの自動販売機でスイムキャップを手に入れることも教えない。心配だった。
```

六月26日
```
フランスの友達と約束したので今日の日本語レッスンが終わったらすぐ電車で大森に向かった

大森に到着、駅の中央改札口から出て昔よく見てた光景が目に入ってくる。まあまあ懐かしい、ここに一年ぐらい住んだから、駅の北口に向かってちょっと歩くと途中でAlexと会った。
ちょっと挨拶して話してまず一緒に回転寿司を食べることにした。店に行く途中と店で食事をしていた時、いろんなことについて話した。
私は自分が最近水泳を勉強していることとピアノの練習状況を彼に言った。また体育センターの様子を説明するために携帯に保存したプールの写真を彼に見せた。
彼は昔水泳が上手だったらしい、昔大雨降っていた時も故郷にある有名な川で泳いだことを教えてくれた。
水泳の話が終わったら、彼は初心者に合うピアノの曲も教えてくれた、春という曲とか練習すれば私はさらに上手になれる。
```
