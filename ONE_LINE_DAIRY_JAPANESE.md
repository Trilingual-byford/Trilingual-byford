６月６日
```
せっかく日曜日なのでカラオケに行くと決めたんけど、午後買い物終わってからBANBANのカラオケ店に行くと「申し訳ない、今満室になっています、これから一時間後部屋が空きかもしれません」と言われた。
まあ、いいか、先買った野菜とか持ってカラオケ室に入ってちょっと恥ずかしいと思うので直ちに帰ると決めました。
歩行で住むところに到着。時間的余裕なのでご飯を作ります。さしぶりに火を使って料理を作るのでまずは簡単な、昔からよく作ったトマトと卵の炒めを作ってみます。
日本の中華料理屋で食べたマトと卵の炒めはネギが使わなかった、私の経験を元にすると葱はマトと卵の炒めの決め手を言っても過言ではない。今回は葱を最後に入れてそうめんと混ぜって食べました。
この程度の努力でこんな美味しさの料理を作って満足した。
```

6月13日
```
土曜日の夜、昼寝をしていた時、急にお母さんからビデオ通話の請求が来ました、何が発生したかと思いながら携帯のオーケーボタンを押してビデオ通話に出ました。
向こうはお母さんとお姉さんの顔が映っています、みんな心配な顔をしていた。お母さんは「祖母の健康状況はかなり厳しい、今日は最後のチャンスかもしれません、祖母に何か言いたいことが有れば、
今のうちに」と言った。
恥ずかしいことかもしれませんが、祖母の病弱な顔が携帯に映されていた時、僕の頭が真っ白で何を言うが全然わからなかった。人って死んでいる人に対して何を言っていいですか？いってらっしゃいとか言ったら
不孝と思われますか、その場で悲しい気持ちとかありませんでした。
晩ご飯はラメンーにすると決めたので、駅付近のラメンー屋に行く途中、田舎で祖母が植えたメロンを食べたあの夏を思い出した、祖母はとても誠実で勤勉な人です。毎回田舎に帰ったこと、祖母と祖父が植えた
新鮮な果物と野菜を楽しむことができます、歳をとったとしても休まない、美味しい果物と野菜を植えて鴨とか鶏とかを飼育する。この勤勉さは誰にも敵わない
祖母は苦痛なくあの世界に行くように。
```
