修正版：
```
日本語で日記を書き始めた！これから１日一回か二日に一回書くかまだ決めていないけど、日本語をうまく使えるようになるまで日記を書き続けたいと思います。
最近は忙しかった。引っ越してから手料理を一回も作らなかった。
この前に買ったホットクックもキッチンの空間を占めてるだけ、シェアハウスなのに無用な物で大切なチッキンの空間を無駄使いにして、申し訳ない気持ちになりました。
これからとにかく考えずに米を買う。ライスを消耗するために料理を作る気になるだろうか？
```
６月６日
```
せっかく日曜日なのでカラオケに行くと決めたんけど、午後買い物に行ってBANBANのカラオケ店に行くと「申し訳ない、今満室になっています、これから一時間後部屋が空きかもしれません」と言われた。
まあ、いいか、先買った野菜とか持ってカラオケ室に入ってちょっと恥ずかしいと思うので直ちに帰ると決めました。
歩行で住むところに到着。時間的余裕なのでご飯を作ります。さしぶりに火を使って料理を作るのでまずは簡単な、昔からよく作ったトマトと卵の炒めを作ってみます。
日本の中華料理屋で食べたマトと卵の炒めはネギが使わなかった、私の経験を元にすると葱はマトと卵の炒めの決め手を言っても過言ではない。今回は葱を最後に入れてそうめんと混ぜって食べました。
この程度の努力でこんな美味しさの料理を作って満足した。
```
